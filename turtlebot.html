Hand Gesture Controlled Turtlebot 4

As part of a my graduate course, I had the oppurtunity to work with a Turtlebot4 Lite. As part of the course project we were a team of two. We decided we wanted to use the Turtlebot4 as a cobot. We wanted to use hand gestures based on the American Sign Language to move and control the Turtlebot4. The Turtlebot4 platform comes with a depth camera. We could use to detect hand gestures. Hence, I wrote code for hand gesture recognition based movement of the Turtlebot4. We used Google Mediapipe python library to detect the hand gesture using specific hand landmarks that are available within the Mediapipe library. I used OpenCV and a RandomForrestClassifier AI model that we trained to recognize the hand gestures and move the Turtlebot4. The whole project was built on the ROS2 Galactic platform.
